## ðŸ”¹ What are Conditional Statements?

Conditional statements let your program make decisions and execute code **based on conditions**.

```python
if condition:
    # code block
elif another_condition:
    # another block
else:
    # fallback block
```

## ðŸ”¹ The `if-elif-else` Ladder

### âœ… Syntax:

```python

    # block 1
elif condition2:
    # block 2
elif condition3:
    # block 3
else:
    # fallback block
```

### âœ… Example:

```python

    print("Grade: A")
elif score >= 80:
    print("Grade: B")
elif score >= 70:
    print("Grade: C")
else:
    print("Grade: F")
```

### âœ… Example:

## ðŸ”¹ Common Errors & Pitfalls

|Mistake|Example|Error|
|---|---|---|
|Missing `:`|`if x > 5`|`SyntaxError: expected ':'`|
|Indentation Error|`if x > 5:\nprint("Yes")`|`IndentationError`|
|Wrong logic with `and/or`|`if x > 5 or < 10`|`SyntaxError` â€” use `if x > 5 or x < 10`|
|Comparing strings with wrong case|`"yes" == "Yes"`|`False` â€” Use `.lower()`|
|Using assignment instead of comparison|`if x = 5:`|`SyntaxError` â€” use `==`|

## ðŸ”¹ Nested Conditionals

You can place `if` inside another `if`.

```python
age = 20
citizen = True

if age >= 18:
    if citizen:
        print("Eligible to vote")
    else:
        print("Not a citizen")
else:
    print("Too young")
```

## ðŸ”¹ One-Liner `if` (Ternary Operator)
